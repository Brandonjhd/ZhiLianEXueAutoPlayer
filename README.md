## 🇨🇳 简体中文

智联E学自动刷课助手 —— 智能高效课程自动化辅助

### 简介

智联E学自动刷课助手是为智联招聘E学(course.zhaopin.com)平台开发的自动化刷课脚本工具。通过自动识别课程目录、智能遍历与播放，帮助用户高效完成课程进度、节省时间，专注于真正的学习内容。

### 主要功能

- ✅ **自动遍历目录** - 智能识别所有章节及小节课程并自动切换
- ✅ **自动点击课程** - 支持一键切换并播放课程，直到最后一课
- ✅ **进度判断跳过** - 自动跳过100%已完成课程，仅播放未完成部分
- 🎬 **自动播放视频** - 针对未完成课程自动完整播放
- 📱 **悬浮球启动** - 美观浮动球入口，UI简洁现代
- 📝 **状态进度显示** - 右下状态框实时显示当前刷课状态
- 🛡️ **安全防护** - 仅在course.zhaopin.com课程页面生效，无数据收集

### 安装使用

#### 1. 安装浏览器扩展

- [Tampermonkey](https://www.tampermonkey.net/) (推荐)
- [Violentmonkey](https://violentmonkey.github.io/)
- [Greasemonkey](https://www.greasespot.net/)

#### 2. 安装脚本

点击下方链接安装脚本：
- [从 GitHub 安装](https://github.com/Brandonjhd/ZhiLianEXueAutoPlayer/raw/main/zhi-lian-E-xue-auto-player.user.js)

#### 3. 使用步骤

1. 打开智联E学课程学习页面（[https://course.zhaopin.com/\*](https://course.zhaopin.com/\*)）
2. 点击页面右下角的紫色悬浮球 🎓
3. 在控制面板选择起始课程
4. 点击「🚀 开始刷课」按钮
5. 脚本会自动切换并播放所有未完成课程

### 技术特点

- 🚀 **极致轻量** - 纯JavaScript实现，无依赖
- 🔒 **安全可靠** - 仅作用于目标课程页面
- 🎨 **友好界面** - 浮动学术帽球与美观操控面板
- ⚡ **智能跳课** - 跳过已完成百分百的课程

### 适用范围

- ✅ 智联E学主站（course.zhaopin.com）
- ✅ 含章节分节课程列表的在线课程

ZhiLian E-Learning Auto Player — Intelligent Automated Learning Aid (Tampermonkey script)

### Introduction

ZhiLian E-Learning Auto Player is designed for course.zhaopin.com. It traverses and plays through all unfinished lessons for the user automatically.

### Features

- ✅ **Auto course traversal** — Detects chapters and lessons, switches and plays automatically
